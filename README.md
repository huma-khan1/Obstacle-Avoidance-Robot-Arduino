# Obstacle Avoidance Robot using Arduino

This project is an autonomous robot that detects and avoids obstacles using an ultrasonic sensor and Arduino.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Battery

## Working Principle
The ultrasonic sensor measures the distance between the robot and nearby objects.  
When an obstacle is detected within a certain distance, the Arduino processes the sensor data and changes the motor direction to avoid collision.

## Applications
- Autonomous robotics
- Smart vehicles
- Industrial automation
